# Telegram Shop Bot

بوت متجر إلكتروني تيليجرام مبني بـ Python + SQLite ويعمل بنظام النقاط.

## المميزات
- تسجيل وتنبيه الأدمن عند دخول عضو جديد مع الاسم واليوزر والايدي.
- لوحة مستخدم: المنتجات، الطلبات، الرصيد، طلب شحن نقاط، التواصل مع الإدارة.
- لوحة إدارة: إدارة المنتجات، الأقسام، الطلبات، شحن نقاط يدوي، إذاعة، إحصائيات.
- حالات الطلب: انتظار، تجهيز الطلب، تم التسليم، اكتمل، ملغي.
- إمكانية الرد على طلب محدد من الأدمن.
- ملاحظات من العميل على الطلب.
- تسليم تلقائي للطلبات للمنتجات التي لها مخزون أكواد.
- تخزين محلي باستخدام SQLite بدون الحاجة إلى MySQL.

## التشغيل
```bash
python -m venv venv
source venv/bin/activate   # في ويندوز: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# عدّل القيم داخل .env
python bot.py
```

## ملاحظات مهمة
- هذه النسخة تعتمد polling لذلك تعمل على أغلب الاستضافات التي تدعم Python 3.10+.
- لا يمكن ضمان أنها تعمل على **كل** استضافة حرفياً بدون أي تعديل، لكن تم تصميمها لتكون بسيطة ومتوافقة قدر الإمكان.
- التسليم التلقائي يحتاج إضافة المخزون من لوحة الأدمن، كل عنصر في سطر منفصل.
- المنتج التلقائي إذا نفد مخزونه يتم إنشاء الطلب بحالة "تجهيز الطلب" ليعالجه الأدمن.
